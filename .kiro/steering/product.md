# Product Overview

leanfmt は Lean 4 向けの高速で意見の強いコードフォーマッタです。CLI から入力された Lean ソースを一貫したスタイルで整形し、開発・CI に統一された整形結果を提供します。

## Core Capabilities

- Lean 4 のネイティブパーサに基づく信頼性の高い整形
- 標準出力 / インプレース / チェックの 3 モード対応
- stdin からの入力にも対応しツール連携が容易
- エラーメッセージ付きで失敗を明確化

## Target Use Cases

- ローカル開発時の Lean 4 ソース整形
- CI でのフォーマットチェックによるスタイル統一
- CLI を他ツールやスクリプトに組み込むワークフロー

## Value Proposition

- Lean の構文解析を直接利用するため、整形の信頼性が高い
- 意見の強いルールでチーム内のスタイルを揃えやすい
- Lake によるビルドと単純な CLI で導入が簡単
